# Spatial distribution of Mediterranean vegetation (co-occurrence matrix)

A dataset containing the abundance of each of the 3,697 species in each
of the 715 sites.

## Usage

``` r
vegemat
```

## Format

A co-occurrence `matrix` with sites as rows and species as columns. Each
element of the matrix represents the abundance of the species in the
site.
