# Spatial distribution of Mediterranean vegetation (sf data.frame)

A dataset containing the geometry of the 715 sites.

## Usage

``` r
vegesf
```

## Format

A `sf` `data.frame` with 715 rows and 2 columns:

- Site:

  Unique site identifier

- geometry:

  Geometry of the site
